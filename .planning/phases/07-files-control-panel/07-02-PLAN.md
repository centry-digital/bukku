1. **Create custom location tools** (`src/tools/custom/location-tools.ts`):
   Follow the pattern from contact-archive.ts. Import McpServer, z, BukkuClient, transformHttpError, transformNetworkError, log.

   Export function `registerLocationTools(server: McpServer, client: BukkuClient): number` that registers 3 tools:

   a) **get-location** tool:
      - Name: "get-location"
      - Description: "Get a location for multi-branch accounting by ID"
      - Input: `{ id: z.number().describe("The location ID") }`
      - Handler: `client.get(\`/location/\${params.id}\`)` — NOTE: singular /location/ path, NOT /locations/
      - Return JSON.stringify(result, null, 2) as text content
      - Standard error handling (transformHttpError/transformNetworkError with tool name "get-location")

   b) **update-location** tool:
      - Name: "update-location"
      - Description: "Update an existing location for multi-branch accounting"
      - Input: `{ id: z.number().describe("The location ID"), data: z.record(z.string(), z.unknown()).describe("Updated data") }`
      - Handler: `client.put(\`/location/\${params.id}\`, params.data)` — singular /location/ path
      - Return JSON.stringify(result, null, 2) as text content
      - Standard error handling with tool name "update-location"

   c) **delete-location** tool:
      - Name: "delete-location"
      - Description: "Delete a location for multi-branch accounting. API may restrict deletion if location is referenced by transactions. Archive instead if deletion fails."
      - Input: `{ id: z.number().describe("The location ID") }`
      - Handler: `await client.delete(\`/location/\${params.id}\`)` — singular /location/ path
      - Return success message: "Successfully deleted location with ID ${params.id}"
      - Standard error handling with tool name "delete-location"

   Log each registration. Return 3.

   Add JSDoc: "Custom location tools for get, update, and delete operations. These use the singular /location/{id} API path instead of the factory's /locations/{id}. The Bukku API has an inconsistency where list/create use /locations but get/update/delete use /location (singular)."

2. **Create control panel archive tools** (`src/tools/custom/control-panel-archive.ts`):
   Follow the pattern from product-archive.ts (which registers 4 tools). Import same dependencies.

   Export function `registerControlPanelArchiveTools(server: McpServer, client: BukkuClient): number` that registers 6 tools:

   a) **archive-location** — PATCH `/location/${params.id}` with `{ is_archived: true }`
      Description: "Archive a location (hide from active lists). Use this for locations referenced by transactions instead of deleting them."

   b) **unarchive-location** — PATCH `/location/${params.id}` with `{ is_archived: false }`
      Description: "Unarchive a location (restore to active lists)."

   c) **archive-tag** — PATCH `/tags/${params.id}` with `{ is_archived: true }`
      Description: "Archive a tag (hide from active lists). Use this for tags referenced by transactions instead of deleting them."

   d) **unarchive-tag** — PATCH `/tags/${params.id}` with `{ is_archived: false }`
      Description: "Unarchive a tag (restore to active lists)."

   e) **archive-tag-group** — PATCH `/tags/groups/${params.id}` with `{ is_archived: true }`
      Description: "Archive a tag group (hide from active lists). Use this for tag groups that contain tags or are referenced by transactions instead of deleting them."

   f) **unarchive-tag-group** — PATCH `/tags/groups/${params.id}` with `{ is_archived: false }`
      Description: "Unarchive a tag group (restore to active lists)."

   All tools: Input `{ id: z.number().describe("The [entity] ID") }`, standard error handling, log registration. Return 6.

   IMPORTANT PATH NOTES:
   - Location archive uses singular `/location/{id}` (consistent with other location single-item operations)
   - Tag archive uses `/tags/{id}` (consistent with tag single-item operations)
   - Tag group archive uses `/tags/groups/{id}` (consistent with tag group single-item operations)
