**# Bukku MCP Server**
One-liner: MCP server that lets Claude read and write accounting data in Bukku.

**## Quick Start**
Numbered steps:
1. Clone and install
2. Build
3. Get your Bukku API token (link to Bukku docs or describe: Control Panel -> Integrations -> API Access)
4. Configure Claude Desktop (show exact JSON for claude_desktop_config.json):
   ```json
   {
     "mcpServers": {
       "bukku": {
         "type": "stdio",
         "command": "node",
         "args": ["/absolute/path/to/bukku-mcp/build/index.js"],
         "env": {
           "BUKKU_API_TOKEN": "your-token-here",
           "BUKKU_COMPANY_SUBDOMAIN": "your-subdomain"
         }
       }
     }
   }
   ```
5. Restart Claude Desktop
6. Try it: "List my sales invoices from last month"

**## Configuration**
Table of environment variables:
| Variable | Required | Description |
|----------|----------|-------------|
| BUKKU_API_TOKEN | Yes | Your Bukku API access token |
| BUKKU_COMPANY_SUBDOMAIN | Yes | Your company subdomain (e.g., 'mycompany' from mycompany.bukku.my) |

- `npm run build` - Compile TypeScript
- `npm test` - Run tests
- `npm start` - Start server (requires env vars)

**## Troubleshooting**
Common issues:
- "Configuration Error" on startup -> check env vars are set
- "Token validation failed" -> check your API token is valid and not expired
